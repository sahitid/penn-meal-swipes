╔══════════════════════════════════════════════════════════════════════╗
║                   MATCHING ALGORITHM LOGIC                            ║
╚══════════════════════════════════════════════════════════════════════╝

For each new form submission:
    │
    ├─► Load all existing submissions of opposite type
    │   (Offers look at Requests, Requests look at Offers)
    │
    ├─► For each potential match, calculate score:
    │   │
    │   ├─► Availability Match (40%)
    │   │   • Count overlapping time slots
    │   │   • Higher score = more overlap
    │   │
    │   ├─► Dining Hall Match (30%)
    │   │   • Count common dining halls
    │   │   • At least 1 shared hall required
    │   │
    │   └─► Interest Similarity (30%)
    │       • Compare text in "About Me" fields
    │       • Count common keywords
    │
    ├─► Sort matches by score (highest first)
    │
    └─► If top match score >= 60%:
        │
        ├─► Send match notification emails
        │
        └─► Record in Matches sheet
            • Match date
            • Both students' info
            • Match score
            • Status: Pending

╔══════════════════════════════════════════════════════════════════════╗
║                      USER JOURNEY                                     ║
╚══════════════════════════════════════════════════════════════════════╝

UNDERCLASSMAN JOURNEY:
━━━━━━━━━━━━━━━━━━━━━━
1. Sees Instagram post → Clicks link in bio
2. Lands on website → Clicks "Offer Meal Swipes"
3. Fills form (2 minutes) → Submits
4. Receives confirmation email
5. Gets matched within 48 hours → Receives intro email
6. Emails upperclassman → Coordinates time
7. Meets at dining hall → Uses meal swipe
8. Enjoys conversation → Makes new connection
9. Receives feedback survey
10. Signs up again for another match!

UPPERCLASSMAN JOURNEY:
━━━━━━━━━━━━━━━━━━━━━━
1. Hears from friend → Visits website
2. Clicks "Request a Meal"
3. Fills form → Submits preferences
4. Receives confirmation email
5. Gets matched → Receives intro with underclassman info
6. Responds to underclassman's email
7. Confirms time/place
8. Shows up at dining hall → Enjoys free meal
9. Shares advice/experience
10. Requests another match!
